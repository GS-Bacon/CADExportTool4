# トラブルシューティング

## よくある問題と解決方法

### SolidWorksに接続できない

**症状**: エクスポート時に「SolidWorksに接続できません」エラー

**解決方法**:
1. SolidWorksが正しくインストールされているか確認
2. SolidWorksを一度手動で起動して正常に動作するか確認
3. 管理者権限でCADExportToolを実行

### ファイルが追加できない

**症状**: ドラッグ＆ドロップしてもファイルがリストに追加されない

**解決方法**:
1. ファイル拡張子が `.SLDDRW`, `.SLDPRT`, `.SLDASM` であることを確認
2. ファイルが他のアプリケーションで開かれていないか確認

### エクスポートに失敗する

**症状**: 一部または全てのファイルのエクスポートが失敗

**原因と対策**:
- **ファイルが開けない**: ファイルが破損していないか確認
- **出力先に書き込めない**: 出力先フォルダの権限を確認
- **SolidWorksバージョン**: 古いバージョンで作成されたファイルは対応していない場合あり

### アプリが起動しない

**症状**: EXEをダブルクリックしても何も起きない

**解決方法**:
1. .NET 8.0 Runtimeがインストールされているか確認
2. [.NET 8.0 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)をダウンロードしてインストール

### 処理が遅い

**原因と対策**:
- **大量のファイル**: ファイル数が多いと時間がかかります
- **複雑なアセンブリ**: 部品点数が多いアセンブリは変換に時間がかかります
- **ネットワークドライブ**: ローカルドライブにコピーして実行

## エラーの報告

問題が解決しない場合は、[Issues](https://github.com/GS-Bacon/CADExportTool4/issues)で報告してください。

報告時に含めてほしい情報:
- エラーメッセージの全文
- 再現手順
- SolidWorksのバージョン
- Windowsのバージョン
