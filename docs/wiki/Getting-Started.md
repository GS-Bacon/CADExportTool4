# 基本的な使い方

## 起動

`CADExportTool.WPF.exe`を実行してアプリを起動します。

## ファイルの追加

### 方法1: ドラッグ＆ドロップ

エクスプローラーからCADファイルを直接アプリウィンドウにドラッグ＆ドロップします。

### 方法2: ファイル追加ボタン

1. 「ファイル追加」ボタンをクリック
2. ファイル選択ダイアログでCADファイルを選択
3. 「開く」をクリック

## 対応ファイル形式

| 拡張子 | ファイルタイプ |
|--------|----------------|
| .SLDDRW | 図面ファイル |
| .SLDPRT | パーツファイル |
| .SLDASM | アセンブリファイル |

## エクスポート

### 1. 出力フォーマットを選択

ファイルタイプに応じて利用可能なフォーマットが異なります:

**図面ファイル (.SLDDRW)**
- PDF
- DXF

**パーツ/アセンブリファイル (.SLDPRT, .SLDASM)**
- IGES
- STEP
- 3MF

### 2. 出力先を設定

出力先オプション:
- **元ファイルと同じ場所**: CADファイルと同じフォルダに出力
- **サブフォルダ**: CADファイルのフォルダ内のサブフォルダに出力
- **カスタムフォルダ**: 指定したフォルダに出力

### 3. エクスポート実行

「エクスポート」ボタンをクリックして変換を開始します。

進捗バーで処理状況を確認できます。

## ファイルの削除

- **選択削除**: ファイルをチェックして「選択削除」ボタン
- **全削除**: 「クリア」ボタンで全ファイルを削除

## キャンセル

エクスポート中に「キャンセル」ボタンをクリックすると処理を中断できます。
