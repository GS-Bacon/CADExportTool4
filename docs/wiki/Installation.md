# インストール

## 動作要件

- **OS**: Windows 10/11
- **ランタイム**: .NET 8.0 Runtime
- **CADソフト**: SolidWorks 2020以降（インストール済み）

## インストール手順

### 1. ダウンロード

[Releases](https://github.com/GS-Bacon/CADExportTool4/releases/latest)から最新版のZIPファイルをダウンロードします。

### 2. 解凍

ダウンロードしたZIPファイルを任意の場所に解凍します。

推奨パス:
- `C:\Program Files\CADExportTool4\`
- `C:\Tools\CADExportTool4\`

### 3. 起動

`CADExportTool.WPF.exe`をダブルクリックして起動します。

## アップデート

アプリ内で新しいバージョンが検出されると通知が表示されます。
「更新」ボタンをクリックすると自動的にダウンロード・更新が行われます。

## アンインストール

解凍したフォルダを削除するだけで完了です。
レジストリや設定ファイルは使用していません。
