# Implementation Plan Command

仕様書から実装計画を作成します。

## Instructions

以下の手順で実装計画を作成してください：

1. **仕様書の確認**
   - `specs/` ディレクトリから最新の仕様書を読み込む
   - 仕様書がない場合は `/speckit.specify` を先に実行するよう案内

2. **技術調査（Phase 0）**
   - 既存コードベースを調査
   - 影響を受けるファイルを特定
   - 技術的な不明点を `[NEEDS CLARIFICATION]` としてマーク
   - 調査結果を `specs/<feature>/research.md` に記録

3. **設計（Phase 1）**
   - アーキテクチャ設計を決定
   - 必要に応じてデータモデルを定義
   - API/インターフェース設計
   - `specs/<feature>/plan.md` に計画を記録

4. **Constitution確認**
   - `memory/constitution.md` の原則との整合性を確認
   - 違反がある場合は justification を記載
